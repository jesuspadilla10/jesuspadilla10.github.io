Portafolio

En esta seccion, les comparto informacion sobre algunos de los proyectos de los que he participado y de los que mas he aprendido a lo largo de mi carrera. Se pretende que en el futuro esta seccion incluya muchos mas proyectos y aplicaciones.

Proyecto PrestaMax

Proyecto desarrollado en lenguaje Java, donde el programa consistio en el control de prestamos y de otros movimientos que realizan los clientes en los bancos.

Base de datos Seguimiento de egresados

Proyecto desarrollado que consistio en una base de datos sobre el Control de egresados del ITCJ, donde se aplicaron conocimientos de interfaces graficas de usuario en Java y de bases de datos usando el gestor de base de datos MySQL.

Control de nominas

Proyecto disenado con la interfaz grafica de Visual Studio, programado en C# y conectado al gestor de bases de datos Postgresql. El proyecto consistio en desarrollar un software para el registro y control de empleados de una empresa, ademas de que se busco imprimir la nomina de algun empleado seleccionado.

GraficaR!

Proyecto desarrollado en el lenguaje C#, donde el proyecto consistio en importar archivos de tipo Excel y CSV, y graficar los datos. El proyecto se buscaba desarrollarse para el personal docente, con el objetivo de graficar los datos de las preguntas de los examenes realizados por sus alumnos.

Sitio web esports

Sitio web desarrollado en HTML y CSS. Consistio en un sitio informativo sobre un equipo de Esports con el objetivo de aplicar los conocimientos aprendidos de programacion web.

Aplicacion movil con Firebase

Aplicacion desarrollada en el software Android Studio. Es el ejemplo de una aplicacion con un Login, donde se registran los usuarios en la base de datos de Firebase de Google. Se aplican conocimientos aprendidos de Java o Kotlin y XML para el diseno de la aplicacion.
